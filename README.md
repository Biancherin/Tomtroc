## Tomtroc

## Pour utiliser ce projet :

- Commencer par cloner le projet.
- installez le projet chez vous, dans un dossier exécuté par un serveur local (type MAMP, WAMP, LAMP, etc...)
- Une fois installé chez vous, créez un base de données vide appelée : "tomtroc"
- Importez le fichier tomtroc.sql dans votre base de données.

## Importer les utilisateurs dans la base de données Tomtroc :

- Ouvrir un terminal et le diriger vers le serveur local du projet.
- Pour importer les utilisateurs, écrivez sur votre terminal seed_users.php

## Lancez le projet !

Pour la configuration du projet, vérifiez dans le fichier config.php les informations de connexion de votre base de données.
